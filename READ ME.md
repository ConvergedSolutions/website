# Converged Solutions Website

A static, responsive website for **Converged Solutions**, an ICT services company based in Lilongwe, Malawi.  
This site showcases the company’s profile, services, clients, and contact information in a clean, modern layout.

---

## Features

- **Responsive Navigation Bar**
  - Includes Home, About Us, Services, Clients, and Contact sections.
  - Hamburger menu toggle for mobile devices.

- **Carousel (Home Section)**
  - Auto-sliding carousel with company highlights.
  - Navigation arrows for manual slide control.

- **Content Sections**
  - **About Us**: Company background and expertise.
  - **Services**: ICT services offered (System Development, Cybersecurity, Data Analysis, etc.).
  - **Clients**: Trusted organizations served.
  - **Contact Form**: Email form that sends queries to `converged_solutions@outlook.com`.

- **Contact Form**
  - Simple form with name, email, and message fields.
  - Uses `mailto:` action to send messages directly via email.

- **Footer**
  - Displays company email and location.
  - Copyright notice.

---

## Technologies Used

- **HTML5** for structure
- **CSS3** for styling and responsive design
- **JavaScript** for:
  - Carousel auto-slide functionality
  - Navigation switching between sections
  - Mobile menu toggle

---

## How to Use

1. Clone or download the repository.
2. Open the `index.html` file in any modern browser.
3. Navigate through sections using the top menu.
4. On mobile devices, use the hamburger menu for navigation.
5. Use the contact form to send inquiries.
